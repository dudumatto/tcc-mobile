# TCC Mobile

App Flutter mobile para o projeto TCC.

## Estrutura

- `lib/core`: API, auth, configuração, tema e utilitários.
- `lib/models`: entidades de domínio.
- `lib/services`: acesso a dados e integrações.
- `lib/providers`: estado global com `ChangeNotifier`.
- `lib/screens`: telas do aplicativo.
- `lib/widgets`: componentes reutilizáveis.
- `test`: testes de widget iniciais.

## Ambiente

- Copie `.env.example` para `.env`
- Defina `API_URL`
